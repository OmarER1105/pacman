# Juego de Pacman 

Este es un juego de Pacman implementado en Python utilizando el módulo `turtle` para la interfaz gráfica y `freegames` para algunas funciones auxiliares.

## Características 

- Pacman se mueve en una cuadrícula y su objetivo es comer todos los puntos blancos.
- Los fantasmas se mueven de manera aleatoria por el mapa.
- El juego termina cuando un fantasma atrapa a Pacman.
- El puntaje del juego se incrementa cada vez que Pacman come un punto blanco.

## Cómo jugar 

- Usa las teclas de flecha para cambiar la dirección de Pacman.
- Intenta comer todos los puntos blancos para aumentar tu puntaje.
- Evita a los fantasmas para no perder el juego.

## Dependencias 

Este juego requiere los siguientes módulos de Python:

- `random`
- `turtle`
- `freegames`
